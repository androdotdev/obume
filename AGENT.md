# Obume

## Stack
Next.js 16 · Drizzle · PostgreSQL · Cloudinary · Tailwind CSS · pnpm

## Structure
```
app/actions/     # Server actions (auth, works, upload)
app/             # Pages, layouts
db/schema.ts     # Schema (users, works)
lib/db.ts        # Drizzle client
```

## Commands
`pnpm dev` · `pnpm build` · `pnpm db:push`
